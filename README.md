# 商品列表（仿京东淘宝等）
## 实现方式

* 左侧为recyclerview
* 右侧为recyclerview
* 通过点击左侧item刷新右侧adapter的数据。
达到二级关联效果。

## 依赖项目有
* RecyclerView通用适配器：https://github.com/CymChad/BaseRecyclerViewAdapterHelper
* 流标签控件：https://github.com/hongyangAndroid/FlowLayout
